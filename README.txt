This apartment finder tool is a webscraping application to find and price match apartments across several rental platforms

Place all of the files under the "apartmentFinder" folder in the same directory with an environment with the required dependencies.

Run "compare.py"

Enter the city and state all lowercase without any special characters.



The two websites currently being used use virtually the same information in the same order. Although they are separate webistes, they seem to be under the same control.

More websites will soon be added to see how far this extends.
